# From One to Many: Few-Shot Deep Ensembles for Slow DoS Attack Detection

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

This repository contains the code and resources for the research paper "From One to Many: Few-Shot Deep Ensembles for Slow DoS Attack Detection". The proposed $MoAE^{2}$ (Mixture of Autoencoder Experts) framework tackles the challenge of detecting stealthy, low-rate Denial-of-Service (DoS) attacks by integrating unsupervised and supervised learning under label-scarce conditions.

## Key Features

* **Few-Shot Learning for Cybersecurity:** Capable of detecting stealthy and evasive Slow DoS attacks using an ensemble strategy combined with a few-shot learning framework;
* **Hybrid DL Architecture:** Constructs an ensemble of unsupervised autoencoder-based detectors, combined through a supervised Mixture of Experts strategy;
* **Lightweight Ensemble ($MoAE^{2}$):** Leverages a single Variational Autoencoder (VAE) model with varying threshold levels to create multiple base detectors of different sensitivities, ensuring a lightweight and computationally efficient solution;
* **Focal Loss Integration:** Effectively addresses the class imbalance typical of intrusion detection datasets by emphasizing learning from rare and difficult-to-detect attack instances;
* **Benchmark-Ready:** Evaluated on the CIC-IDS2017 dataset, significantly outperforming classical supervised ML baselines (such as XGBoost, KNN, Random Forest, and Decision Tree) under few-shot constraints.

## Framework Structure

Instead of training multiple independent models, the framework is structured around the following core components:

* **Unsupervised Base Detectors:** A Variational Autoencoder (VAE) trained exclusively on legitimate network traffic to extract general-purpose representations and model normal behavior;
* **Threshold-based Ensemble:** A set of detectors generated by applying different thresholds to the reconstruction error of the single pre-trained VAE;
* **Neural Gate (Mixture of Experts):** A trainable feed-forward neural network that acts as a gating network, assigning a relevance score to each base detector to combine their outputs.
